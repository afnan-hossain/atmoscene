# Contributing to Atmoscene

Thank you for helping build a more complete, accessible, and globally useful weather-visualization library.

## Before contributing

1. Read [TASK.md](TASK.md) for the current phase and acceptance criteria.
2. Keep icon ids semantic and stable.
3. Do not introduce duplicate suns, moons, or eclipses in composed scenes.
4. Preserve edge feathering, reduced-motion behaviour, attribution, and format parity.
5. Include the relevant tests and visual-review material with each contribution.

Detailed development setup and asset-authoring instructions will be added when the Phase 01 monorepo scaffold is complete.

## Support the project

Code, design review, documentation, testing, translation, and issue triage are all valuable contributions. If you prefer to support ongoing development financially, visit [Buy Me a Coffee](https://buymeacoffee.com/afnan.hossain).
