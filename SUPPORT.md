# Support Atmoscene

Atmoscene is being built as a free, open-source weather icon and atmospheric scene system.

## Project help

- Use GitHub Issues for reproducible bugs and focused asset requests.
- Use GitHub Discussions for integration questions, ideas, and community showcases once Discussions is enabled.
- Read the documentation before opening an implementation question.

## Support development

If Atmoscene saves you time or improves a product, you can support continued design, animation, testing, and documentation work:

[Support Atmoscene on Buy Me a Coffee](https://buymeacoffee.com/afnan.hossain)

Financial support is optional and never required to use, contribute to, or request improvements for the project.
