GETTING STARTED

Use one living scene.
Load only what you need.

Atmoscene’s browser release provides 3,552 time-aware animated SVG scene combinations through a shared renderer and category-based catalogue. Open a focused family, choose day or night, then load only the scene required by an interface.

USAGE

Use the browser scene renderer

const renderer = window.AtmosceneSceneRenderer;
const scene = {
  mode: 'night',
  phase: renderer.nightPhase('midnight', 'full'),
  weather: renderer.weather('night', 'aurora-strong')
};
target.innerHTML = renderer.sceneMarkup(scene, 1);

The catalogue uses the same public semantic keys, making it a visual reference for selecting the correct weather and celestial state.

FRAMEWORKS

Framework-neutral scene markup

The renderer returns SVG-based HTML markup that can be mounted from plain JavaScript or wrapped by React, Vue, Svelte and Web Components. Category routing prevents an interface from bundling or rendering the complete catalogue.

DELIVERY

Cacheable and on demand

/api/v1/scenes/by-id/day.sunset.none.rain.json

Use the public versioned API to fetch one scene by stable semantic ID, browse family collection endpoints for discovery, and load the zero-dependency SDK only when an interface needs to mount a scene. See the complete API v1 reference.

API REFERENCE

Semantic scene resolution

renderer.weather(mode, weatherKey)
renderer.dayPhase(phaseKey)
renderer.nightPhase(timeKey, moonPhaseKey)
renderer.sceneMarkup(scene, uniqueId)

Every rendered scene exposes semantic data attributes for family, weather, time phase, moon phase, wind profile, visibility, precipitation and special effects. These attributes can drive theming, accessibility descriptions and provider adapters without changing the artwork.

QUICK ANSWERS

Animated weather asset questions

What is an animated weather SVG?

An animated weather SVG is a scalable vector weather graphic whose motion runs in the browser without a raster video. It stays sharp at different sizes and can be loaded as a standalone asset.

When should I use SVG instead of Lottie?

Use animated SVG for lightweight web delivery and simple embedding. Use Lottie when an application needs timeline control or a native playback runtime. Atmoscene plans both formats from one canonical motion source.

How will Atmoscene map Open-Meteo or WMO weather codes?

The provider adapter will normalize a weather code, day or night state, daylight phase, moon phase, wind, visibility and severity. The core resolver will then return one semantic Atmoscene scene ID without tying the library to a single data provider.

Does an app need to render all 3,552 scenes?

No. Category pages show six representative scenes and the catalogue renders at most 24 previews per page. A weather interface needs only the scene selected for its current state.
