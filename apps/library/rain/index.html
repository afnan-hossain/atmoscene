PRECIPITATION RESOLVER

Animated rain SVG for changing precipitation

A useful weather interface must distinguish drizzle from sustained rain, intermittent showers and heavy precipitation. Atmoscene varies drop density, fall speed, cloud weight, wind shear and scene tint while retaining the correct daylight or moon state.

Rain remains an independent compositor layer. The same system can resolve ordinary rain, tropical showers, monsoon conditions and storm precipitation without duplicating the cloud or celestial artwork.

InputRate + probability + phase

Precipitation intensity, weather code, local sky period and wind establish the scene.

ComposeDrops + cloud + shear

Density and fall speed change with intensity while direction responds to wind.

OutputReadable rain scene

Animated SVG keeps drops legible at card size and protects interface content zones.
