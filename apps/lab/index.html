Built for weather interfaces

What is Atmoscene?

Atmoscene is a free, open-source animated weather icon and scene library. It combines reusable weather SVG artwork with time-of-day gradients, wind, visibility, precipitation, moon phases and celestial events, so a weather app can show one coherent sky instead of stacking unrelated symbols.

Animated weather SVG

Lightweight browser-ready motion for clear skies, clouds, rain, storms, fog, haze, wind, moon phases, aurora, tropical weather and celestial events, with reduced-motion behavior built in.

Weather scenes, not only icons

Daylight phase, local night, lunar state and atmospheric layers resolve into a single scene with themeable surfaces for dashboards, websites and mobile interfaces.

Developer-ready formats

Semantic scene IDs and transparent SVG layers can be used from plain JavaScript or wrapped for React, Vue, Svelte and Web Components without rendering the full catalogue.

Looking for implementation details? Read the animated weather asset documentation, integrate a stable scene ID through the public API, browse the curated asset collections, or refine combinations in the Scene Explorer.
